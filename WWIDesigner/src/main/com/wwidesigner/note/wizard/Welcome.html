In the context of WWIDesigner, a Tuning represents the notes (symbols and frequencies) and associated fingerings (open and closed holes) for a particular instrument. This wizard guides you in the specification of a Tuning and all of the optional persistable, reusable components that make up that Tuning.

The Scale Symbols Page supports the specification of the Scale Symbol Set, the arbitrary symbols that identify each note. Each symbol may be any length and content. On this page, you may create a set de novo, use as a starting point a number of common Western symbol sets, or load one from a saved file. And the output from this page can be saved to a file and/or used in subsequent pages. A valid Scale Symbol Set, to be saved, must contain a name and at least one symbol; to be used in the Scale with Intervals Page, it does not need a name.

The Musical Temperament Page supports the specification of the Temperament, the intervals that separate the notes of a scale - defined as a frequency ratio relative to the root note. As in the prior page, you may create de novo, load from a couple of Western-standard, or load from file the initial Temperament, saving and/or using the Temperament. A valid Temperament must contain, to be saved, a name and at least one ratio; to be used in the Scales with Intervals Page, it does not need a name.

The Scale with Intervals Page supports the specification of a Scale with Intervals, which is the merging of a Scale Symbol Set and a Temperament (Intervals). Both the Symbols and Temperament can be loaded from their respective page or from a file, created de novo (but not saved), and modified. The Scale with Intervals may be created de novo, or by dragging and dropping selected symbols and ratios from the surrounding respective tables. Currently, a Scale with Intervals can not be persisted, only used in subsequent pages. A valid Scale with Intervals contains at least one note with both a symbol and ratio.

The Scale with Frequencies Page supports the specification of a Scale with Frequencies, which is a set of notes each defined by a symbol and a frequency (in Hz). A Scale with Frequencies may be created in a number of ways: De novo, entering values directly in the Notes table; by loading an existing Scale file and optionally modifying it; or by calculating the frequencies based on the intervals in a Scale with Intervals and a reference note in that scale with a specified frequency.
